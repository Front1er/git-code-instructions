## 1 git native code versioning

### 1.1 git user name and email

```cmd
git config --global user.name "Your Name"
git config --global user.email "youremail@yourdomain.com"
```

confirm information:

```cmd
git config --list
user.name=Your Name
user.email=youreamil@yourdomain.com
```
